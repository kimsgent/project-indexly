## Release v2.0.1 (2026-04-05)

### Changes
- fix(workflow): update publish pipeline to create a PR for Homebrew formula updates instead of pushing directly to main
- fix(workflow): add pull-requests write permission for the publish workflow token
- fix(workflow): add fallback to push release/brew-<tag> branch when GitHub Actions cannot create PRs
- chore(release): harden release automation to comply with protected-branch PR-only rules
