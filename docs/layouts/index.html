Index and Search, Fast and Local

Project Indexly gives you lightning-fast file indexing with full-text search and custom tagging. Lightweight, offline, and built for privacy.

  • ✅ Tag and group files your way
  • ✅ Search by name, content, or tag
  • ✅ Export results to Markdown or CSV
  • ✅ Works fully offline on your system

Analyze & Infer — Structured Insights

Beyond indexing, Indexly can analyze and summarize a wide range of file formats — CSV, JSON, NDJSON, XLSX, XML, YAML, Parquet, and SQLite. For CSV datasets, run rigorous statistical inference directly in the CLI.

  • 📊 Analyze multiple formats: CSV, JSON, NDJSON, XLSX, XML, YAML, Parquet, SQLite
  • 📈 Perform statistical inference on CSV datasets: correlation, t-tests, ANOVA, regression
  • 🔁 Automatic routing: assumption-aware test selection for reliable results
  • 📄 Export analysis & inference reports to Markdown or PDF
  • 💡 Quick insights: summary statistics, validation, and preview per file

AutoDoctor — Windows Diagnostics and Remediation

AutoDoctor is a local-first Windows diagnostics platform that combines PowerShell automation, SQLite telemetry history, a FastAPI API, and a live dashboard. It helps users detect root-cause issues quickly, monitor system health over time, and apply reliable remediation steps with full visibility.

  • 🩺 Root-cause health scoring across CPU, memory, disk, network, and event signals
  • 🗃️ Local SQLite storage for diagnostics, alerts, remediation logs, and telemetry
  • 📡 Built-in API and dashboard for real-time monitoring and support workflows
  • 🛠️ Optional self-healing actions for common Windows reliability problems
